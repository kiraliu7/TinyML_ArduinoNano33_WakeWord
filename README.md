# TinyML_ArduinoNano33_WakeWord
Team MissingPort: Bocheng Wang, Yihai Long, Zhengtong Liu, Zhenyang Lin

The name of each directory indicate the wake words its model recognize. For example, "wakeWord_left_right" would respond to "left" and "right"

To deploy the model, simple upload everything in the directory onto the board via the Arduino IDE
